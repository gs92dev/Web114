  2. Identify three distinct "best practice" programming practices and explain how they create better code.
    1- The use of ";" makes the code more readable and less error-prone. With that, it is easier to identify where the statement is finishing.
    2-The use of const instead of var and let makes it easier to identify variables that should not be changed. In case another developer working on the same team, for example, tries to modify it, they will get an error.

  5. Explain the phrase "Fractions divided by powers of 2 are exactly represented in the binary system" and what universal problem this is referring to in all programming languages. (You might want to view my YouTube on why 0.1 + 0.2 is not equal to 0.3). Another way to think about this is to ask yourself these questions -- can all decimal integers (the numbering system we use that uses values of 0-9) be accurately represented by the binary (the numbering system the computer uses with values of 0 and 1)? Can all decimal fractions be accurately represented by the binary numbering system? Which ones can? Which ones cannot?
    Binary (base-2) accurately represents fractions divisible by powers of 2 (e.g., 0.5, 0.25). This is because they can be expressed as sums of negative powers of 2. However, fractions like 0.1 cannot be represented exactly due to binary limitations. These often lead to repeating binary patterns.
