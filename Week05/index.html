Questions

  1. What is the difference between a function declaration and a function expression?

    Function declaration is a statement that defines a function with the specified parameters and statements. It is hoisted to the top of the scope in which it is defined.
    A function expression is a function that is assigned to a variable. It is not hoisted to the top of the scope in which it is defined.
  2. Why do we create functions in the first place? There are two big reasons. One has to do with WHEN the statements in a function run. The other has to do with how often the statements in a function are run.

    We usually create functions to run a block of code when we need it to run(call or invoke the function) or in other words, the piece of code is going to execute only when we want to execute it.
    We can run the same block of code multiple times without having to write it out each time. That saves time and makes our code easier to maintain and debug.
